# Trabalho Final - Qualidade de Software (QoS)

## Estrutura do Projeto

Este repositório contém a documentação de testes do Sistema SAFE (Sistema de Apoio à Biossegurança em Ambientes Físicos). Os **documentos originais** em PDF e LaTeX fornecidos pelo LENS estão na pasta `Docs Originais/`. Estes foram convertidos para **formato Markdown** (`*.md` na raiz) para facilitar edição, colaboração e interação com LLMs, mantendo a estrutura e conteúdo completos dos planos e casos de teste.

Os **documentos finais para entrega** estão na pasta `LATEX/`, formatados profissionalmente seguindo os templates do LENS. Compile os arquivos `.tex` com `pdflatex` para gerar os PDFs finais. A pasta raiz também contém `Requisitos.md` e `Informações e variáveis.md` como documentos de apoio contendo especificações do sistema SAFE.
